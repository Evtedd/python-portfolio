Telegram Join Request Bot
Telegram бот для автоматического отбора пользователей перед выдачей доступа в закрытый канал и чат. Пользователь проходит тест, бот считает баллы, проверяет критичные вопросы и выдаёт доступ при успешном результате. Администратор может просматривать статистику, пользователей, ответы, логи и вручную управлять доступом.
Возможности
 Для пользователя
* Запуск теста через `/start`.
* Тест из 6 основных вопросов.
* Условный дополнительный вопрос.
* Система баллов.
* Критичные вопросы, без которых доступ не выдаётся.
* Автоматическая выдача доступа при успешном прохождении.
* Одна повторная попытка при неудаче.
* Продолжение с прерванного вопроса после рестарта.
* Сообщения с результатом прохождения.
Для администратора
* Админ панель через кнопку при `/start`.
* Общая статистика.
* Список пользователей с пагинацией.
* Поиск пользователей.
* Карточка пользователя.
* Просмотр ответов пользователя.
* Просмотр причин отказа.
* Логи событий.
* Логи конкретного пользователя.
* Статистика источников трафика по UTM/source меткам.
* Ручная выдача доступа.
* Ручной отказ.
* Сброс попытки.
* Блокировка пользователя.
* Управление администраторами для super admin.
* Настройка ссылок на канал и чат.

## Логика прохождения

* Максимальный балл: `9`.
* Проходной балл: `7`.
* Максимум попыток: `2`.
* Некоторые вопросы помечены как критические.
* Если критичный вопрос провален, автоматический доступ не выдаётся даже при достаточном количестве баллов.

## Хранимые данные

Бот хранит:
* Telegram user id.
* Имя пользователя.
* Статус прохождения.
* Источник трафика.
* Сессии теста.
* Ответы на вопросы.
* Логи действий.
* Список администраторов.
* Настройки ссылок.
